MyFlix

A web application that will provide users with access to information about different movies, directors, and genres.
Users will be able to sign up, update their personal information, and create a list of their favorite movies

Endpoints

Endpoint URL:

/movies/genre/[Name]

Description:

Retrieves data for a single genre in the registry by genre name

Method:

GET

Expected data format:


None

Expected response format:


JSON object containing data on a movie genre (Name, Description).


Example:

{
Name: "Action",
Description: "Physical action takes precedence in the storytelling. The film will often have continuous motion and action including physical stunts, chases, fights, battles, and races. The story usually revolves around a hero that has a goal, but is facing incredible odds to obtain it"

Endpoint URL:

/users

Description:

Add a new user to the registry with a unique indentifier

Method:

POST

Expected data format:


JSON object holding data about a user to add.


Example:

{
_id: "001",
Username: "MrMovie",
Password: "IL0v3F1lm5",
Email: "mrmovie@gmail.com",
Birthday: "05/06/1984"
}

Expected response format:


JSON object holding data about a user to add.


Example:

{
_id: "001",
Username: "MrMovie",
Password: "IL0v3F1lm5",
Email: "mrmovie@gmail.com",
Birthday: "05/06/1984"
}

Endpoint URL:

/users/[Username]

Description:

Update a user's information by Username

Method:

PUT

Expected data format:


JSON object holding data about a user to add.


Example:

{
_id: "001",
Username: "MrMovie",
Password: "IL0v3F1lm5",
Email: "mrmovie@gmail.com",
Birthday: "05/06/1984"
}

Expected response format:


Text response to notify the user the update was successful

Endpoint URL:

/users/[Username]/movies/[_id]

Description:

Add a favourite movie to the user's favourite list

Method:

POST

Expected data format:

None

Expected response format:


Text response to notify the user the movie has been added to their favourites list

Endpoint URL:

/users/[Username]/movies/[_id]

Description:

Remove a favourite movie from the user's favourite list

Method:

DELETE

Expected data format:

None

Expected response format:


Text response to notify the user the movie has been removed from their favourites list

Endpoint URL:

/users/[Username]

Description:

Remove a user from the registry

Method:

DELETE

Expected data format:

None

Expected response format:


Text response to notify the user they have successfully removed their profile
